Advanced Topics

This section has tutorials for three subjects:

  • Address Tables: how to format a table of code or data addresses easily.
  • Extension Scripts: an introduction to scripts that automate formatting of inline data.
  • Visualizations: converting graphical data to bitmap or wireframe images that can be embedded into the disassembly listing.

These features are not essential, but they can make your life easier. All of the tutorials assume you are already familiar with SourceGen.
